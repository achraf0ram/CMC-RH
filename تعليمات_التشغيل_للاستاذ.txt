تعليمات تشغيل مشروع إدارة الموارد البشرية (Gestion RH)
====================================================

مرحباً أستاذي الكريم،

هذا الملف يحتوي على تعليمات مفصلة لتشغيل المشروع لأول مرة. المشروع يتكون من جزئين:
1. الواجهة الأمامية (Frontend) - React + TypeScript
2. الخادم الخلفي (Backend) - Laravel + PHP

متطلبات النظام:
================
- Node.js (الإصدار 18 أو أحدث)
- PHP (الإصدار 8.1 أو أحدث)
- Composer (مدير حزم PHP)
- MySQL أو MariaDB
- Git

الخطوة الأولى: إعداد قاعدة البيانات
====================================

1. قم بإنشاء قاعدة بيانات جديدة في MySQL:
  

2. انتقل إلى مجلد الخادم:
   cd server

3. انسخ ملف البيئة:
   cp .env.example .env

4. قم بتثبيت حزم PHP:
   composer install

6. أنشئ مفتاح التطبيق:
   php artisan key:generate

7. قم بتشغيل الهجرات (Migrations):
   php artisan migrate

8. قم بتشغيل البذور (Seeders) لإنشاء المستخدم الإداري:
   php artisan db:seed

الخطوة الثانية: إعداد الواجهة الأمامية
=======================================

1. انتقل إلى مجلد العميل:
   cd client

2. قم بتثبيت حزم Node.js:
   npm install

3. انسخ ملف البيئة للواجهة الأمامية:
   cp .env.example .env

4. عدّل ملف .env وأضف عنوان الخادم:
   VITE_API_URL=http://localhost:8000

الخطوة الثالثة: تشغيل المشروع
==============================

1. شغّل الخادم الخلفي (في مجلد server):
   php artisan serve
   
   سيتم تشغيل الخادم على العنوان: http://localhost:8000

2. في نافذة طرفية جديدة، شغّل الواجهة الأمامية (في مجلد client):
   npm run dev
   
   سيتم تشغيل التطبيق على العنوان: http://localhost:5173

الخطوة الرابعة: تسجيل الدخول
==============================

بعد تشغيل المشروع، يمكنك تسجيل الدخول باستخدام:

معلومات المستخدم الإداري:
- البريد الإلكتروني: admin@example.com
- كلمة المرور: password

معلومات المستخدم العادي:
- البريد الإلكتروني: user@example.com
- كلمة المرور: password

ملاحظات مهمة:
==============

1. تأكد من أن خدمة MySQL تعمل على جهازك
2. تأكد من أن المنافذ 8000 و 5173 غير مستخدمة
3. إذا واجهت مشاكل في التثبيت، تأكد من تحديث Composer و npm
4. للمساعدة الإضافية، راجع ملف README.md في كل مجلد

ميزات المشروع:
===============

- نظام تسجيل دخول وخروج
- إدارة الطلبات (إجازات، شهادات عمل، أوامر مهمة)
- نظام إشعارات في الوقت الفعلي
- واجهة إدارية متكاملة
- دعم اللغة العربية والإنجليزية
- نظام محادثات عاجلة
- إدارة الملفات والوثائق

في حالة وجود أي استفسارات أو مشاكل، لا تتردد في التواصل معي.
